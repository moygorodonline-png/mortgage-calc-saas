## 📋 О проекте

**SaaS-платформа** для сайтов застройщиков, которая превращает обычный ипотечный калькулятор в мощный инструмент сбора лидов.

### 🎯 Ключевые отличия:

| Обычные калькуляторы | Наш калькулятор |
|----------------------|-----------------|
| Посмотрел → закрыл | Посмотрел → оставил заявку |
| Ознакомительная функция | Инструмент лидогенерации |
| Нет интеграций | CRM + Email + Telegram |

### 🔥 Фишки:
- **Горячие лиды:** форма заявки (имя + телефон) → автоматически в CRM
- **Холодные лиды:** отправка расчёта на email/Telegram
- **Актуальные ставки:** централизованное обновление данных банков
- **Скрытые цены:** калькулятор работает, конкуренты не видят цены
- **Полный контекст:** вся информация о расчёте передаётся в CRM

## 🛠️ Технологии

- **Backend:** PHP 8.2+, Laravel 11
- **Database:** MySQL 8.0
- **Frontend Widget:** Vanilla JavaScript (ES6+)
- **Admin Panel:** Laravel Blade + Alpine.js
- **Server:** VPS Beget, Nginx
- **Integrations:** Bitrix24, Google Sheets, Email (SMTP)
