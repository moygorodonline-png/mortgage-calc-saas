## [0.0.1] - 2025-01-28

### Added
- 📋 Создана дорожная карта проекта (ROADMAP.md v2.0)
- 🤖 Создан файл CLAUDE.md - инструкция для AI-помощников
- 📊 Создан трекер прогресса PROGRESS.md
- 📝 Создан CHANGELOG.md
- 📁 Создана структура папок проекта
- 📄 Шаблон для session notes

### Changed
- ♻️ Переработана концепция продукта: от WP-плагина к SaaS-решению
- 🏗️ Определена архитектура: PHP Backend + JS Widget

### Context
- Первая сессия планирования проекта
- Определено позиционирование: не просто калькулятор, а инструмент лидогенерации
- Выбрана модель монетизации: внедрение + подписка

### Decisions
- ✅ SaaS-модель вместо WordPress-плагина
- ✅ Laravel для backend
- ✅ Vanilla JS для виджета (без фреймворков)
- ✅ Инкост остаётся на текущем коде (полигон для идей)
- ✅ Google Sheets как первая интеграция CRM
